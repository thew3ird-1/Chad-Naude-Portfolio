# Chad Denzel Naude Portfolio

This repository contains the source code for the personal portfolio of **Chad Denzel Naude**. The project is a modern, responsive web application built using **React**, **TypeScript**, and **Vite**, and is deployed on [Vercel](https://chad-naude-portfolio.vercel.app).

## üõ†Ô∏è Getting Started

### Prerequisites

* [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/thew3ird-1/Chad-Naude-Portfolio.git
cd Chad-Naude-Portfolio

```


2. **Install dependencies:**
```bash
npm install

```


3. **Environment Setup:**
Set your `GEMINI_API_KEY` in a `.env.local` file to enable AI-powered features (if applicable).

### Development

Run the application locally in development mode:

```bash
npm run dev

```

The application will be available at `http://localhost:5173`.

### Building for Production

To create an optimized production build:

```bash
npm run build

```
